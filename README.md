### This is Ernie Su's quadopter project.

I write this blog to record my processing.

First, I use the follwing stuff in my quadcopter:
* F330 Frame
* YH2212 Motor x 4
* 8045 Propellers x 4
* ESC x4
* UBEC
* B6AC (Charger)
* 3S, 4000mah Battery
* Pixhawk 2.4.8
* M8N GPS
* PPM Encoder
* 6S FLY-SKY Telecontroller

I'll keep my update on my blog.
