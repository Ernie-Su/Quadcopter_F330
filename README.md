### This is Ernie Su's quadopter project.

I use the follwing stuff in my quadcopter:
* F330 frame
* YH2212 motor x4
* 8045 Propellers x4
* ESC x4
* UBEC ()
* B6AC (Charger)
* 3S, 4000mah Battery
* Pixhawk 2.4.8
* M8N GPS
* PPM Encoder

I'll keep my update on this blog.
